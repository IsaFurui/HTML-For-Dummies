Tags

Tudo que está entre os caracteres “<” e “>”, servem para informar ao navegador a estrutura e o significado do conteúdo. Existe um conjunto de tags padronizado pelo W3C, logo não é possível criar próprias tags HTML.

As tags devem ser obrigatoriamente fechadas com o mesmo nome do início da tag + “/” antes de seu nome.

A tags que fogem desse padrão como a tag de quebra de linha <br/>, que usa o “/” dentro da tag, porém a quebra de linha se tornou obsoleta no HTML5.

Estrutura Raíz de uma Página Web

Tag Raíz - HTML

Raiz do documento que deve ser aberta e fechada. É possível utilizar dentro da tag raiz o atributo lang, como atributo global, para definir o idioma da página.

Cabeçalho - head

Onde contém as tags para manter o documento em ordem. Dentro dela, temos as tags:

Tag Para que serve
Title
 Título do documento, sendo visualizada na barra de título do navegador
Meta
 Engloba as metainformações, como descrição da página, palavras-chave, etc
Script
 Responsável pela inclusão ou definição de scripts relacionados ao documento
Link
 Responsável pela inclusão das folhas de estilo externas relacionadas ao documento. Possibilita também a inclusão de favicons (pequenos ícones que aparecem na barra de endereços do navegador)
Style
 Também é responsável pelo vínculo das folhas de estilo quando elas são declaradas diretamente no documento

Corpo - body

Responsável pela estruturação do documento, seu conteúdo e sua apresentação.


Elementos Básicos

Cabeçalho - header

Fica na página superior da página, onde se agrupa o título da página, sua logo, etc.

Barra de Navegação - nav

Podendo ser tanto na horizontal quanto na vertical, contém os links de navegação para as páginas/seções do site.

Conteúdo - main

Conteúdo principal da página que pode ser dividido em seções (section).

Barra Lateral - aside

Pode conter tanto links de navegação quanto informações para complementar o conteúdo principal

Rodapé - footer

Área final da página, onde são colocados links, avisos de direitos autorais, informações sobre o proprietário da página e informações sobre o site.


Tipo e Composição das Tags

Os tipos de tags podem ser: estruturais, textuais e semânticas. Além disso, as tags podem ser atributos, estes são compostos por nome e um valor.

<img src=”imagem.jpg” alt=”minha imagem”/>

A tag image possui os atributos src e alt. O atributo src (source) define o endereço e o nome da imagem. Já o atributo alt define o que será exibido no navegador caso a imagem não carregue.

Atributos Mais Importantes

Os atributos mais importantes para desenvolver uma página são:

  1. ID: Utilizado para definir um identificador único para tag de um documento.
  2. Class: Pode ser usado em uma ou mais tags, com essa identificação, é possível fazer referência a um mais atributos para inserir estilização visual nas páginas através de CSS e interação através de JavaScript.

Tags de Formatação

Apesar de toda formatação ser feita no CSS, há algumas tags direcionadas a formatação visual do conteúdo no HTML (que ainda podem ser substituídas no CSS):

Tag Para que serve
<b>
 De bold, aplica o efeito de negrito
<strong>
 Equivalente ao negrito e o marca como importante.
<i>
 De italic, aplica o efeito de itálico.
<em>
 Equivalente ao itálico e marca como importante em um texto.
<u>
 De underline, aplica o texto de sublinhado.
<mark>
 Se refere a um texto marcado.
<del>
 Risca o elemento ao meio. Ex: elemento
<sup>
 Deixa o texto sobescrito. Ex: 22
<sub>
 Deixa o texto subescrito. Ex: exemplo
<abbr>
 usado com siglas, quando se passa o mouse sobre a sigla, o significado da sigla aparece em uma caixa. Ex: ONU
<small>
 Define um texto pequeno. Ex: texto pequeno
<font>
 Define as características da fonte, podendo alterar cor, tamanho e tipo da fonte. Ex: texto pequeno
